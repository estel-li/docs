## 安装与使用 


:::steps
### 1. 获取项目

```bash
git clone https://github.com/estle-li/estel_docs.git
```

::tip
或者，您也可以从 [GitHub 仓库](https://github.com/estle-li/estel_docs) 克隆或下载该模板。
::

### 导航到项目目录

```bash
cd [project-name]
```

### 编译项目

::::code-group

```bash [pnpm]
pnpm i && pnpm dev
```

```bash [yarn]
yarn install && yarn dev
```

```bash [npm]
npm install && npm run dev
```

```bash [bun]
bun install && bun dev
```
::::


::tip
做得好！浏览器窗口应会自动打开，访问 http://localhost:3000。
::
